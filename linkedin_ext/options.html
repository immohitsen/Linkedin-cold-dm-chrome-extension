LinkedIn Cold DM Generator

Version 1.1  ·  Powered by Groq & Llama 3

Features

  • ⚡ AI-generated, personalized DMs in seconds
  • 🎯 Scrapes LinkedIn profile data automatically
  • ✍️ Choose tone, length, and reason for reaching out
  • 📨 Injects DM directly into the LinkedIn message box

How It Works

1
Open a LinkedIn profile and click the message button.
2
Click the extension icon. Choose tone, length, and add context.
3
Hit Generate & Send DM — the message is written for you instantly.
4
Review, edit if needed, and hit Send on LinkedIn.

No API key needed  ·  Your data never stored
